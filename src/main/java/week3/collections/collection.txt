Definition:
A java Collection is a data structure

A data structure is data organized in a particular way plus a set of operations to perform on these data.perations 
such as read, insert, update, delere, sort etc

Generally speaking, there is no such thing as the best data structure, 
because sepecific data structures perform better than others depending on the situation.

A Java collection is comprised of objects of the same type (Same as in Arrays)

MAPS
************

-A map is a set of ordered pairs, each pair contains a key and a value. In a map, all keys are required to be unique, the values can be repeated.

-Maps have three primary methods: one methode to add a specified key-value pairs, one to retrieve the value for a given key, one to remove a key -value pair from the map

- Maps are very useful when we need to search by a key.
An Entry is a (key, value) pairring.
